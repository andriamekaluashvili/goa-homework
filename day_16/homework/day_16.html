HTML

HyperText Markup Language or HTML
is the standard markup language
for documents designed to
be displayed in a web browser.

CSS

Cascading Style Sheets (CSS) is a style
sheet language used for specifying
the presentation and styling of
a document written
in a markup language such as HTML

JS

JavaScript often abbreviated as JS,
is a programming language that
is one of the core technologies of
the World Wide Web,
alongside HTML and CSS.
